1. backend
1.1. Tạo package.json
    -   npm init -y

1.2. Cài các dependencies cần thiết
    -   npm i express dotenv mongoose jsonwebtoken stripe cloudinary cookie-parser bcryptjs ioredis nodemon
    * cloudinary sẽ bị deprecated -> npm install cloudinary@latest

1.3. Truy cập upstash.com để dùng các dịch vụ Database as a Service (DBaaS)

1.4. Cài lib để dùng Upstash Redis
    -   npm install @upstash/redis

1.5. Truy cập https://dashboard.stripe.com/test/dashboard để lấy STRIPE_SECRET_KEY

1.6. Cài gói cấu hình để BE cho phép truy cập từ nhiều nguồn
    -   npm i cors

1.7. Cài cross-env giúp thiết lập biến môi trường (environment variables) theo cách cross-platform
    -   npm i -D cross-env

2. frontend
2.1. Tạo project react/vite
    -   npm create vite@latest .

2.2. Cài các dependencies cần thiết
    -   npm install

2.3. Cài thêm dêpndencies
    (framer-motion: animation, react-confetti: bắn hiệu ứng pháo giấy, react-hot-toast: thông báo, recharts: thư viện vẽ biểu đồ cho React)
    -   npm i react-router-dom axios @stripe/stripe-js framer-motion lucide-react react-confetti react-hot-toast recharts zustand
